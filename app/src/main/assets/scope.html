Variable Scope 


Local Variable:



1. A variable that is declared inside method is local variable.
2.Access Modifiers cannot be used with them.
3.No default values.



Instance Variable:



1. A variable that is declared inside the class but outside the 
method is instance variable.
2.Not declared as static.
3. They are created with the use of new keyword.
4.Access Modifiers can be given.
5.They have default values.



Class/Static Variable:



1.A variable that is declared as static is static variable.
2.It cannot be local.
3.They have static keyword.
4.Stored in static memory.


Example:  
class A{
int data=50;//instance variable
static int m=100;//static variable
void method(){
int n=90;//local variable
}
}
