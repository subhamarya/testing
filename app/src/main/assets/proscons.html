Pros and Cons


Pros:



    1. Java is simple.
	2. Java is object oriented.
	3. Java is platform independent.
	4. Java is Secure.
	5. Java is Multithreaded.
	6.Java is Robust.



Cons:



    1. Java is slower and more memory-consuming than C or C++.
    2. No local constants.
    3. Java is predominantly a single-paradigm language.
